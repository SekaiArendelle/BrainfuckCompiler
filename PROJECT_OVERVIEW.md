# Brainfuck LLVM 编译器项目概览

## 项目简介

这是一个完整的Brainfuck语言编译器项目，使用LLVM作为后端生成高效的原生机器码。项目采用现代C++17编写，具有清晰的架构和完善的注释。

## 核心特性

### 🎯 完整语言支持
- 实现全部8条Brainfuck指令
- 标准内存模型（30,000字节数组）
- 指针边界检查
- 标准I/O支持

### ⚡ 高性能编译
- LLVM IR代码生成
- 多级别优化支持（-O0到-O3）
- JIT即时执行模式
- 原生机器码输出

### 🔧 开发友好
- 详细的错误报告
- 调试信息生成
- 编译统计信息
- 模块化代码架构

### 🚀 用户友好
- 直观的命令行界面
- 丰富的示例程序
- 完整的文档说明
- 跨平台支持

## 技术架构

### 类结构
```cpp
class BrainfuckCompiler {
    // 核心编译器类
    // - LLVM上下文管理
    // - IR生成
    // - 优化支持
    // - JIT执行
};
```

### 模块划分
- **核心编译器** (`BrainfuckCompiler.*`) - 主要编译逻辑
- **命令行接口** (`main.cpp`) - 用户交互
- **示例程序** (`examples/`) - 测试用例
- **构建系统** (`CMakeLists.txt`, `Makefile`) - 项目构建

### LLVM集成
- **IR生成**: 使用`IRBuilder`创建LLVM指令
- **内存管理**: `AllocaInst`分配运行时内存
- **优化**: 标准LLVM优化Pass
- **执行**: MCJIT/OrcJIT即时编译
